# FoodBridge 🍲 - Hackathon 2026

## 🚀 The Problem
Bengaluru generates massive amounts of daily surplus food, yet NGOs face severe logistical hurdles in locating and claiming it. 

## 💡 Our Proposed System
FoodBridge is a centralized, real-time web portal that connects donors (restaurants/events) directly with NGOs using interactive mapping and secure QR-code verification.

## 🛠 Hardware & Software Requirements
* **Frontend:** Streamlit (Python UI framework)
* **Backend logic:** Python, Pandas 
* **Data Visualization:** Folium (Interactive mapping)
* **Authentication/Security:** Python `qrcode` library
* **Version Control & Deployment:** Git, GitHub, Streamlit Community Cloud
